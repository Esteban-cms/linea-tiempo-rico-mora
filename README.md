# Línea de Tiempo - Proyecto Minero Rico Mora

Este repositorio contiene la línea de tiempo de los eventos más importantes relacionados con el proyecto minero **Rico Mora**, incluyendo interacciones de la compañía con la autoridad ambiental y el registro de títulos mineros.

## Descripción del Proyecto

La compañía **Rico Mora** suscribió el contrato de concesión IH8-10541 para la explotación de minerales en Yalí. A lo largo de los años, se han registrado eventos clave, como suspensiones de obligaciones, solicitudes y otorgamiento de licencias ambientales, informes técnicos y actos administrativos.
